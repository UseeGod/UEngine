# Manual

`Attach` = 후킹 할 앱의 App Name 입력 후 Enter 를 눌러 앱 등록 후 Attach

<img width="1354" alt="스크린샷 2022-04-17 오후 4 14 01" src="https://user-images.githubusercontent.com/67410476/163704657-e16bd26c-8efc-4e0d-a722-0109ff8ead89.png">

`Spawn` = 후킹 할 앱의 Package Name 입력 후 Enter 를 눌러 앱 등록 후 Spawn

<img width="1336" alt="스크린샷 2022-04-17 오후 4 15 07" src="https://user-images.githubusercontent.com/67410476/163704691-7443e567-8122-41aa-a0d6-a6544e77e897.png">

`Upload JS Code` = JS code 업로드 시 JS파일에 작성한 코드를 등록하여 후킹 가능

<img width="705" alt="스크린샷 2022-04-17 오후 4 18 46" src="https://user-images.githubusercontent.com/67410476/163704802-cecc710d-9c92-4555-9c91-1e7184639f60.png">
